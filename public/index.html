As part of my project I worked on the Web App Maker project under professor Jose Annunziato. This web app allows a user to create a web page which can be edited through the app. Web App Maker lets you perform CRUD operations on the fields within each page through a mobile-friendly design.

Application Architecture

Web App Maker follows the MVC architecture pattern where the app is divided into models, views and controllers where models manipulate the data and contain the core app logic, views specify the display of application data and controllers act as a bridge between the views and controllers converting data into formats expected by these components.
